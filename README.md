## About The Project

![Pico Y Placa](docs/assets/empty-app.png)

The project is made to predict the traffic restrictions in Quito

Days based Restrictions:

| Dias      | Dígitos |
| --------- | ------- |
| Lunes     | 1 y 2   |
| Martes    | 3 y 4   |
| Miercoles | 5 y 6   |
| Jueves    | 7 y 8   |
| Viernes   | 9 y 0   |
